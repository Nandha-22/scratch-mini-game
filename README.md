# Scratch Mini Game

This is a small mini game created using Scratch.

## About the Game
- Built using Scratch
- Focuses on basic game logic and events
- Created as a learning project

## How to Play
1. Open the .sb3 file in Scratch
2. Click the green flag
3. Follow on-screen instructions

- Logical thinking
